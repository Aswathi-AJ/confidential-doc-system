# Confidential Government Document Sharing System

A secure web-based system for sharing confidential documents using modern cryptographic techniques.

## Technologies Used

Frontend:
- React

Backend:
- Node.js
- Express.js

Database:
- MySQL

Security:
- Argon2id (password hashing)
- JWT (authentication)
- AES-256-GCM (file encryption)
- Role-Based Access Control

## Features

- Secure user registration and login
- JWT authentication
- Role-based access control
- Secure document upload
- AES-256-GCM encryption for documents
- Secure document download with decryption
